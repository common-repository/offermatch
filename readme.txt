A widget that displays related offers to your pages or posts. 

== Description ==

The offers are selected by matching tags you have assigned to the offer and the tags you have assigned to your content. When there is no offer found you can display a default offer. You can display any number of offers in the widget.

== Installation ==

1. Unzip the contents and upload the entire `offermatch` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance -> Widgets to add the widget to any of the widgets areas. 
4. Configure the number of offers to display and default offer
4. Under "Plugins" find OfferMatch. On this page you can add offers and tag them so they match your posts and pages. The offer description supports HTML.

== Frequently Asked Questions ==

= Can I include affiliate codes, adsense or other special codes? =

Yes, you can place anything there as long as it is valid HTML/Javascript

= What happens if more than one offer matches the tags on a page? =

If the offers that match the tags on the page are more than the number of offers you have defined in the widget settings, the widget will show only the most appropriate offers. It will rank them by the number of matching tags.
